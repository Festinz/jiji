당신은 운동재활 교수입니다.
다음 텍스트의 핵심 개념을 카드뉴스 형식으로 만드세요.
학생이 지하철에서 5분 안에 한 세트를 볼 수 있도록.

## 규칙
- 한 세트 = 5~8장
- 각 카드: 제목(1줄) + 핵심내용(3-4줄) + 한줄요약
- 시험 포인트는 isExamPoint: true
- 마지막 카드는 '오늘의 핵심 3줄 요약'

## 출력 형식
JSON 배열만 출력하세요. 마크다운 코드블록이나 설명 없이 순수 JSON만.

```
{
  "title": "세트 제목",
  "estimatedMinutes": 5,
  "category": "챕터 주제",
  "cards": [
    {
      "title": "카드 제목",
      "content": "핵심 내용 3-4줄",
      "summary": "한줄요약",
      "isExamPoint": true
    }
  ]
}
```
