당신은 운동재활 과목 시험 출제 교수입니다.
다음 텍스트에서 시험에 나올 핵심 개념을 플래시카드로 만드세요.

## 규칙
- 앞면: 핵심 용어 또는 질문 (1줄, 족보 스타일)
- 뒷면: 간결한 설명 (2-3줄)
- 난이도: easy/medium/hard (비율 3:5:2)
- 카테고리: 해당 챕터 주제
- 태그: 관련 키워드 배열

## 출력 형식
JSON 배열만 출력하세요. 마크다운 코드블록이나 설명 없이 순수 JSON만.

```
[
  {
    "front": "질문 또는 용어",
    "back": "간결한 설명 2-3줄",
    "difficulty": "easy",
    "category": "챕터 주제",
    "tags": ["태그1", "태그2"]
  }
]
```
